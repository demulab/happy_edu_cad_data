# Happy Edu CAD Parts

## About Happy Edu
Happy Edu is an open-hardware educational robot developed by the Demura Laboratory.  

This repository contains STEP files for educational CAD components.
These components are provided for reference only, so feel free to customize them to suit your own needs.



## Components List

| File Name | Component Name | Description |
|-----------|----------------|-------------|
| LiDAR_mounting_part.step | LiDAR Mount | Component for mounting a LiDAR sensor |
| U2D2_mounting_part.step | U2D2 Mount | Component for mounting the U2D2 unit |
| arm_mounting_part.step | Arm Mount | Component for mounting an arm assembly |
| camera_mounting_part.step | Camera Mount | Component for mounting a camera |
| emergency_stop_switch_cover.step | Emergency Stop Switch Cover | Cover for the emergency stop switch |
| kachaka_base_extension_part.step | Kachaka Base Extension | Extension part for the Kachaka base |
| microphone_mounting_part.step | Microphone Mount | Component for mounting a microphone |
| pcb_mounting_part.step | PCB Mount | Component for mounting a printed circuit board (PCB) |
| post_mounting_part.step | Post Mount | Component for mounting a post or support structure |
